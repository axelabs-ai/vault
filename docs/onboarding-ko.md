# 우리 가족 비밀번호 금고 — 시작 가이드

안녕하세요. 오늘부터 우리 집 비밀번호를 한 곳에 안전하게 모아둘 거예요.
복잡해 보이지만, 천천히 따라오시면 30분 안에 끝납니다.

이 문서는 IT를 잘 모르셔도 읽을 수 있도록 풀어 썼습니다.
중간에 막히면 언제든 운영자(가족 안에서 이걸 설치해 준 사람)에게 연락하세요.

> 운영자 연락처: 카톡 또는 가족 단톡방에서 "vault 도움" 한 마디면 됩니다.

---

## 1. 시작하기 전에 (5분)

### 지금 우리 집 비밀번호는 어떻게 관리되고 있나요?

잠깐 솔직하게 생각해 봅시다. 지금 네이버, 카카오, 은행, 쇼핑몰 비밀번호를 어디에 저장하고 계신가요?

- 머릿속에 외워 두기 → 자꾸 까먹어서 매번 "비밀번호 찾기" 누르고 계시죠.
- 메모장 앱에 적어 두기 → 폰 분실하면 누가든 다 볼 수 있어요.
- 종이 수첩에 적어 두기 → 외출 중에는 못 봐요.
- 같은 비밀번호를 모든 사이트에 → 한 군데만 털리면 다 털립니다.
- 브라우저 자동 저장 → 노트북 빌려준 사람이 다 봅니다.

이 중 한두 가지에 해당되신다면, 지금 우리 가족 비밀번호는 위험한 상태예요.

### vault가 뭔가요?

vault는 우리 집 안에 설치된 **비밀번호 전용 금고**라고 생각하시면 됩니다.

비유하자면 이렇습니다.
- 지금까지는 비밀번호를 책상 서랍, 냉장고 옆 메모지, 머릿속에 막 두고 살았어요.
- 이제부터는 **현관 비밀번호 하나**(이걸 Master Password라고 부릅니다)만 외우시면,
  그 안에 들어가서 모든 사이트의 비밀번호를 꺼내 쓸 수 있어요.
- 그리고 이 금고는 **우리 집 안에만 있어요**. 외부 회사 서버에 안 올라갑니다.

![](screenshots/onboard-01-concept.png)

### 무엇이 좋아진가요?

- 비밀번호는 한 번만 외우면 됩니다 (Master Password).
- 폰, 노트북 어디서든 같은 비밀번호를 꺼낼 수 있어요.
- 새 사이트 가입할 때 vault가 어려운 비밀번호를 자동으로 만들어 줍니다.
- 가족끼리 공유할 비밀번호(넷플릭스, 쿠팡 등)는 한 번만 등록하면 모두가 봐요.
- 잃어버려도 복구할 수 있는 종이(Recovery Code)가 있어요.

### 준비물 4개

다음 4가지를 챙겨 주세요.

1. **스마트폰** — 아이폰이든 갤럭시든 상관없습니다.
2. **노트북 또는 데스크탑** — 맥, 윈도우 모두 가능합니다.
3. **종이 한 장과 펜** — 비상용 복구 코드를 적어 둘 거예요. 잃어버리면 안 됩니다.
4. **운영자가 보낸 카톡 초대 링크** — 아직 못 받으셨다면 운영자에게 "초대 링크 주세요" 한 마디.

> 다음으로: 카톡으로 받은 초대 링크를 폰이 아닌 **노트북**에서 먼저 여세요.

## 2. 첫 가입 (10분)

### 2.1 초대 링크 클릭

운영자가 카톡이나 이메일로 다음과 같은 링크를 보내 드립니다.

```
https://macmini.<우리집이름>.ts.net/#/register?email=...
```

이 링크를 **노트북 브라우저**에서 클릭하세요.

> 폰에서 먼저 열어도 되지만, Master Password 같은 중요한 비밀번호를 처음 만들 때는
> 큰 화면에서 천천히 보면서 만드시는 게 안전합니다.

![](screenshots/onboard-02-invite.png)

### 2.2 가입 화면

링크를 누르면 가입 화면이 뜹니다. 다음 정보를 입력하세요.

- **이메일** — 이미 채워져 있을 거예요. 그대로 두세요.
- **이름** — 본인 이름(예: 김영희).
- **Master Password** — 이게 가장 중요합니다. 아래에서 자세히 설명합니다.

### 2.3 Master Password 만들기 — 가장 중요한 단계

Master Password는 우리 집 **현관 비밀번호**예요.
이걸 알면 우리 집 안 모든 서랍(=다른 사이트 비밀번호)을 열 수 있어요.
그래서 다음 규칙을 꼭 지켜 주세요.

**좋은 Master Password 만드는 법**

1. **한국어 단어 4개를 짧은 막대(-)로 이어서** 만드세요.
   - 예: `푸른-감자-책상-여덟`
   - 예: `우리집-강아지-콩이-사랑해`
   - 영문자와 숫자 섞을 필요 없습니다. 한국어 단어가 더 외우기 쉽고 더 안전해요.
2. **12자 이상**이어야 합니다. 위 예시처럼 단어 4개면 충분히 넘어갑니다.
3. **다른 곳에서 쓰는 비밀번호와 절대 같으면 안 됩니다.** 새로 만드세요.
4. **본인만 아는 단어 조합**이어야 합니다. 가족 이름·생일·전화번호 금지.

**해서는 안 되는 것**

- 메모장 앱, 이메일, 카톡에 "내 Master Password는 OOO야" 적기 → 절대 금지.
- 다른 사람에게 알려 주기 → 배우자에게도 알려 주지 마세요.
  (공유가 필요한 비밀번호는 5번 섹션의 "가족 공유" 기능을 씁니다.)
- 너무 짧은 비밀번호(예: `1234`, `password`).

![](screenshots/onboard-03-master-password.png)

### 2.4 Recovery Code 인쇄 — 비상용 열쇠

가입이 끝나면 화면에 긴 영문·숫자 조합이 나옵니다.
이게 **Recovery Code**(복구 코드)예요.

예시: `R4-VAULT-7T9X-K2MN-Q8WP-Z1JL`

이건 Master Password를 잊어버렸을 때 쓰는 **유일한** 비상 열쇠입니다.
다음과 같이 보관하세요.

1. 화면을 **그대로 종이에 손글씨로 정확하게 옮겨 적습니다.**
   (사진 찍지 마세요. 폰이 털리면 같이 털립니다.)
2. 종이를 **거실 금고, 책상 서랍 안쪽, 책장 사이** 등 본인만 아는 곳에 보관합니다.
3. 가능하면 **두 장 적어서 다른 곳에 따로** 보관하세요. (예: 한 장은 집, 한 장은 사무실 서랍)
4. **절대 폰 메모, 이메일, 카톡, 클라우드에 저장하지 마세요.**

> Recovery Code를 잃어버리면 Master Password 잊었을 때 **운영자도 복구해 줄 수 없습니다.**
> 그동안 모은 비밀번호가 모두 사라집니다. 꼭 종이에 적으세요.

![](screenshots/onboard-04-recovery.png)

> 다음으로: 운영자에게 "가입 완료, Recovery Code 보관함" 한 마디 보내 주세요.
> 그리고 노트북에 Bitwarden 앱을 깔러 갑시다.

## 3. Mac 또는 노트북에 설치 (5분)

이제 **매번 비밀번호를 자동으로 채워 주는 도우미 앱**을 설치합니다.
이 앱 이름이 **Bitwarden**(비트워든)이에요. vault는 그 앱이 보는 우리 집 금고예요.

### 3.1 자동 설치 스크립트 (가장 쉬운 방법, Mac만)

운영자가 카톡으로 `vault-onboard.sh`라는 파일을 보내 드렸을 거예요.
다음 순서로 하세요.

1. 카톡에서 파일을 받아 **다운로드** 폴더에 저장합니다.
2. 다운로드 폴더에서 파일을 **두 번 클릭**합니다.
3. "확인 안 된 개발자" 경고가 나오면, 우클릭(또는 control+클릭) → **열기** → 다시 **열기**.
4. 검은 창이 뜨면서 자동으로:
   - Bitwarden 앱 설치
   - 우리 집 vault 서버 주소 자동 등록
   - Safari·Chrome 확장 페이지 열림
5. 마지막에 "끝났습니다!" 메시지가 나오면 닫으셔도 됩니다.

![](screenshots/onboard-05-mac-installer.png)

### 3.2 수동 설치 (Windows거나 스크립트 안 될 때)

1. 브라우저로 https://bitwarden.com/download 접속.
2. 본인 운영체제에 맞는 데스크탑 앱 다운로드.
3. 설치 후 앱 실행.
4. **로그인 화면 들어가기 전에**, 좌상단 톱니바퀴 아이콘 → **Self-hosted environment** 선택.
5. Server URL 칸에 다음을 정확히 붙여넣기 (운영자가 카톡으로 보낸 주소):
   ```
   https://macmini.<우리집이름>.ts.net
   ```
6. 저장 → 메인 화면으로 돌아와서 이메일과 Master Password로 로그인.

### 3.3 브라우저 자동 채우기 켜기

이제 Safari나 Chrome에서 비밀번호를 자동으로 채울 수 있게 해 줍니다.

**Chrome (윈도우·맥 공통)**

1. Chrome 열기 → 주소창에 `chrome.google.com/webstore` 입력.
2. "Bitwarden" 검색 → 첫 번째 항목 클릭.
3. "Chrome에 추가" → "확장 프로그램 추가".
4. Chrome 오른쪽 위에 작은 자물쇠 아이콘(파란색)이 생깁니다.
5. 그 아이콘 클릭 → 로그인 화면에서 톱니바퀴 → Self-hosted → 서버 URL 입력 → 로그인.

**Safari (Mac만)**

1. Safari 열기 → 상단 메뉴 Safari → 환경설정 → 확장 프로그램.
2. Bitwarden이 보이면 체크. 안 보이면 App Store에서 "Bitwarden for Safari" 검색해서 설치.
3. 자물쇠 아이콘 → 로그인 (위와 동일).

![](screenshots/onboard-06-extension.png)

> 다음으로: 자물쇠 아이콘이 잠금 해제(unlock)된 상태인지 확인. 그러면 폰 설치로 갑니다.

## 4. 폰에 설치 (10분)

폰에도 같은 Bitwarden 앱을 설치합니다.
그래야 외출 중에도 비밀번호를 꺼낼 수 있어요.

### 4.1 아이폰 (iOS)

1. **App Store** 열기 → "Bitwarden" 검색 → 다운로드(무료).
2. 앱을 실행하면 로그인 화면이 보입니다.
   **로그인 누르기 전에**, 화면 왼쪽 위 톱니바퀴(또는 "지역") 아이콘 누르기.
3. **Self-hosted** 또는 **자체 호스팅 서버** 선택.
4. Server URL 칸에 운영자가 보낸 주소를 정확히 붙여넣기:
   ```
   https://macmini.<우리집이름>.ts.net
   ```
5. 저장 → 메인 화면으로 돌아와서 이메일·Master Password 입력.
6. 손가락(Face ID)으로 잠금 해제 사용하시려면 설정 → **잠금 해제** → Face ID 켜기.

![](screenshots/onboard-07-ios-server.png)

#### 자동 채우기 켜기 (아이폰 — 필수!)

이걸 안 켜면 vault가 있어도 비밀번호 자동 입력이 안 됩니다.

1. 아이폰 **설정** 앱 → 아래로 스크롤 → **암호** 누르기.
2. **암호 옵션** → **암호 자동 완성** 켜기(초록색으로).
3. 아래 목록에서 **iCloud 키체인** 끄기, **Bitwarden** 체크.

![](screenshots/onboard-08-ios-autofill.png)

### 4.2 안드로이드

1. **Play 스토어** 열기 → "Bitwarden" 검색 → 설치(무료).
2. 앱 실행 → 로그인 화면 → 좌상단 톱니바퀴.
3. **자체 호스팅 서버** 선택 → Server URL 붙여넣기:
   ```
   https://macmini.<우리집이름>.ts.net
   ```
4. 저장 → 이메일·Master Password 입력.
5. 지문 잠금 해제 사용하시려면 설정 → 잠금 해제 → 지문 켜기.

#### 자동 채우기 켜기 (안드로이드)

1. 폰 **설정** 앱 → **언어 및 입력** 또는 **일반 관리** → **자동 채우기 서비스**.
2. **Bitwarden** 선택.
3. 사용 허락 팝업 나오면 모두 **허용**.

![](screenshots/onboard-09-android-autofill.png)

### 4.3 Tailscale 앱 설치 (꼭 필요!)

vault 서버는 우리 집 안에만 있기 때문에, 폰이 우리 집 네트워크에 들어와야 보입니다.
그 다리 역할을 하는 게 **Tailscale**이라는 앱이에요.

1. App Store / Play 스토어에서 "Tailscale" 검색 → 설치.
2. 운영자가 보낸 **Tailscale 초대 링크**를 폰에서 누름 → "수락".
3. 앱이 자동 실행되며 "Connected" 표시가 뜨면 끝.
4. 폰을 켜 두는 동안 항상 백그라운드에서 켜져 있어야 합니다.

> Tailscale이 꺼져 있으면 vault 접속이 안 됩니다.
> "Cannot reach server"라는 오류가 뜨면 Tailscale 켜져 있는지 가장 먼저 확인하세요.

> 다음으로: 폰에서 Bitwarden 잠금 해제하고, 가족 컬렉션이 보이는지 확인하세요.

## 5. 첫 사용 (5분)

이제 진짜로 써 봅시다. 세 가지를 차례로 해 볼게요.

### 5.1 네이버 자동 입력 체험

가장 자주 쓰시는 네이버로 연습해 봅니다.

1. Safari 또는 Chrome에서 `naver.com` 접속.
2. 우측 상단 **NAVER 로그인** 버튼 누름.
3. 아이디 칸이나 비밀번호 칸을 클릭하면:
   - 데스크탑: 브라우저 오른쪽 위 작은 파란 자물쇠 아이콘에 빨간 점이 생김 → 누름 → 항목 선택.
   - 폰: 키보드 위에 작은 자물쇠 아이콘 또는 "암호" 버튼이 뜸 → 누름 → 항목 선택 → 지문/Face ID.
4. 아이디와 비밀번호가 자동으로 채워집니다 → 로그인 누름.

![](screenshots/onboard-10-naver-autofill.png)

> 만약 네이버 항목이 vault에 아직 없다면, 6 섹션 "수동 추가"를 보세요.

### 5.2 새 사이트 가입 — 비밀번호 자동 생성

새로 회원가입할 때 vault가 진가를 발휘합니다.
예시로 새로운 쇼핑몰에 가입한다고 칠게요.

1. 쇼핑몰의 회원가입 페이지 열기.
2. 이메일, 이름 등 다른 칸은 평소처럼 본인이 입력.
3. **비밀번호 칸** 클릭하면:
   - 자물쇠 아이콘에 "비밀번호 생성" 또는 "Generate" 버튼이 나타남.
   - 누르면 `8xK#mP2vL@qN5wR` 같은 어려운 비밀번호가 자동 입력됨.
4. 비밀번호 확인 칸도 같은 버튼으로 채움.
5. 가입하기 → 브라우저 자물쇠 아이콘이 "이 비밀번호를 vault에 저장할까요?" 물어봄 → **저장**.

이제 다음에 이 사이트 방문하면 위 5.1처럼 자동 입력됩니다.
이제부터는 **새 비밀번호를 머릿속으로 생각하지 마세요.** vault에 맡기시면 됩니다.

![](screenshots/onboard-11-generate.png)

### 5.3 가족 공유 컬렉션 사용

운영자가 미리 만들어 둔 **"가족 공유"** 컬렉션이 있습니다.
넷플릭스, 쿠팡, 통신사 같은 가족 공동 계정이 여기 들어 있어요.

**컬렉션 보기**

1. Bitwarden 앱 열기 → 하단 **금고(Vault)** 탭.
2. 위쪽 **모든 항목** 옆에 **컬렉션** 메뉴 → **가족 공유** 선택.
3. 우리 가족 공동 계정 목록이 나옵니다.

**내가 새 공동 비밀번호 추가하는 법**

1. 금고 화면 오른쪽 아래 **+ 추가** 버튼.
2. 이름(예: "OO쇼핑 가족 계정"), 사용자명, 비밀번호 입력.
3. **소유자**를 본인이 아닌 **Realchoice Family**로 바꿉니다.
4. **컬렉션**에서 **가족 공유** 체크.
5. 저장 → 가족 모두에게 자동 동기화됨.

![](screenshots/onboard-12-collection.png)

**주의**

- 가족 공유에 추가한 항목은 가족 누구나 보고 수정할 수 있습니다.
- 본인만 쓰는 비밀번호(예: 본인 카카오뱅크)는 **소유자: 나**로 두세요.
  그러면 가족도 못 봅니다.

> 다음으로: 운영자에게 "첫 사용 완료" 한 마디 보내고, 다음 며칠 동안 평소처럼 인터넷 쓰시면서
> 자동 입력이 잘 되는지 체험해 보세요. 안 되는 사이트는 운영자에게 알려 주세요.

## 6. 자주 묻는 질문 (FAQ)

더 자세한 질문 모음은 별도 파일 `faq-ko.md`에 있습니다. 여기서는 가장 자주 나오는 10가지만.

### Q1. Master Password를 잊어버렸어요.

→ Recovery Code 종이를 찾아서 운영자에게 가져 오세요.
   복구 절차를 함께 진행합니다.
   Recovery Code도 잃어버리면 그동안 모은 비밀번호가 모두 사라집니다.

### Q2. Recovery Code 종이를 잃어버렸어요.

→ 운영자에게 즉시 연락. 새로 발급해 드립니다(아직 Master Password를 잊지 않은 동안만 가능).

### Q3. 폰을 잃어버렸어요.

→ 운영자에게 즉시 연락. 잃어버린 폰의 vault 접근을 차단합니다.
   새 폰에 다시 설치하시면 됩니다(섹션 4 다시 따라하기).

### Q4. vault 접속이 안 됩니다 ("Cannot reach server").

→ 폰·노트북의 **Tailscale 앱**이 켜져 있는지 확인하세요.
   꺼져 있으면 우리 집 네트워크에 못 들어와서 vault가 안 보입니다.
   여전히 안 되면 운영자에게 연락.

### Q5. 자동 입력이 안 됩니다.

→ 섹션 4.1·4.2의 **자동 채우기 설정**을 다시 확인하세요.
   많은 분들이 이 단계를 건너뛰셔서 안 되는 경우입니다.

### Q6. 비밀번호를 가족에게 공유하고 싶어요.

→ 섹션 5.3의 **가족 공유 컬렉션** 사용. 절대 카톡·문자·이메일로 보내지 마세요.

### Q7. 2단계 인증(은행에서 받는 6자리 코드)도 vault에 넣을 수 있나요?

→ 네. 항목 편집 → "인증기 키 (TOTP)" 칸 → 사이트가 보여 주는 QR 코드 스캔.
   이후 vault가 6자리 코드를 30초마다 보여 줍니다.

### Q8. 카드번호도 저장 가능한가요?

→ 네. 새 항목 추가 → 유형 **카드** 선택. 다만 운영자와 한 번 상의하세요.

### Q9. 무료인가요?

→ 네. 100% 무료입니다. 우리 집 서버에 직접 설치되어 있어서 매월 요금이 없습니다.

### Q10. 친구·이웃에게도 권유할 수 있나요?

→ 아니요. 우리 가족 vault는 **가족 전용**입니다.
   친구분이 필요하시면 Bitwarden 공식 무료 계정(`bitwarden.com`)을 안내해 주세요.
