# 자주 묻는 질문 — 우리 가족 vault

이 문서는 vault를 쓰시면서 자주 막히는 부분을 모아 둔 곳입니다.
어떤 질문이든 부끄러워하지 마시고 운영자에게 한 마디 주세요.

> 운영자 연락처: 가족 단톡방에 "vault 도움" 또는 직접 카톡.

처음 설치하시는 분은 먼저 `onboarding-ko.md`부터 읽어 주세요.
이 FAQ는 그 다음 단계입니다.

---

## 비상 상황

### Q1. Master Password를 잊어버렸어요.

가장 흔한 비상 상황입니다. 다음 순서로 하세요.

1. **당황하지 마세요.** 시간이 좀 걸릴 뿐, 거의 다 복구됩니다.
2. **Recovery Code 종이**를 찾으세요. 가입할 때 손글씨로 적어 두신 그 종이입니다.
   거실 금고, 책상 서랍, 책장 등을 확인.
3. 운영자에게 연락 → 함께 복구 절차 진행.
4. 새 Master Password를 정한 뒤, **다시 종이에 새 Recovery Code를 적어** 보관.

Recovery Code도 못 찾으시면 그동안 모은 비밀번호가 모두 사라집니다.
이 경우 각 사이트마다 "비밀번호 찾기"로 하나씩 다시 받는 수밖에 없어요.

### Q2. Recovery Code 종이를 잃어버렸어요. Master Password는 기억나요.

지금이 기회입니다. 즉시 다음을 하세요.

1. 데스크탑 브라우저에서 vault 로그인.
2. 우측 상단 본인 이름 → **계정 설정** → **보안** → **Recovery Code 재발급**.
3. 새 코드가 나오면 **종이에 다시 손글씨로** 적기.
4. 안전한 곳(이번엔 두 곳에 나눠서)에 보관.

만약 본인이 못 하시겠으면 운영자에게 연락 주세요.

### Q3. 폰을 잃어버렸어요. 도둑 맞은 것 같아요.

**즉시** 운영자에게 연락하세요. 시간이 곧 보안입니다.

운영자가 할 일:
- 잃어버린 폰의 vault 세션을 강제 종료(차단)합니다.
- 새 폰에 다시 설치할 수 있도록 안내해 드립니다.

본인이 할 일:
- 통신사에 연락해서 폰 분실 신고(유심 정지).
- "내 아이폰 찾기" 또는 "내 디바이스 찾기"로 폰 위치 확인 및 원격 잠금.
- 다음 날까지 새 폰에서 다시 vault 로그인 (섹션 4 그대로).

> 너무 두려워 마세요. Master Password를 모르는 사람은 폰만 있어도 vault를 못 엽니다.

### Q4. 다른 사람 폰에 잠깐 로그인하고 싶어요. (예: 카페에서 가족 폰 빌림)

비추천합니다. 그래도 꼭 필요하시면:

1. 브라우저(Safari/Chrome)에서 vault 주소 직접 입력.
   (단, 그 폰에 Tailscale이 깔려 있어야 함.)
2. 이메일 + Master Password 로그인.
3. **반드시 작업 끝나면 로그아웃** 하고, 브라우저 기록 삭제.

가능하면 본인 폰 빌려 와서 쓰시거나, 다른 방법을 찾으세요.

## 일상 사용

### Q5. vault 접속이 안 됩니다.

가장 자주 있는 원인 순서대로 확인하세요.

1. **Tailscale 앱이 켜져 있나요?** 폰·노트북 모두.
   - 폰: Tailscale 앱 열기 → 상단 토글이 초록색인지.
   - 노트북: 상단 메뉴 바의 Tailscale 아이콘 → "Connected"인지.
2. **Wi-Fi 또는 LTE 연결되어 있나요?**
3. **macmini 본체가 켜져 있나요?** (운영자에게 확인 부탁)
4. **서버 주소를 잘못 입력하지 않았나요?**
   - 정확히 `https://macmini.<우리집이름>.ts.net` 이어야 함.
   - `http://` (s 없음)이면 안 됨.

위 4가지 모두 OK인데 안 되면 운영자에게 연락.

### Q6. 비밀번호를 가족과 공유하고 싶어요.

`onboarding-ko.md` 5.3 섹션을 보세요. 요약하면:

1. Bitwarden 앱에서 항목 추가 또는 기존 항목 편집.
2. **소유자**를 본인이 아닌 **Realchoice Family**로 변경.
3. **컬렉션**에서 **가족 공유** 체크.
4. 저장 → 가족 모두에게 동기화.

**절대 하지 마세요**: 카톡·문자·이메일로 비밀번호 보내기.
한 번이라도 카톡에 보내면 그 비밀번호는 더 이상 비밀번호가 아닙니다.

### Q7. 2단계 인증(6자리 코드)도 vault에서 받을 수 있나요?

네. 은행이나 거래소에서 받는 그 6자리 숫자 코드 말씀이시죠.
다음과 같이 설정하세요.

1. 평소처럼 사이트에서 "2단계 인증 설정" 들어가기.
2. 사이트가 QR 코드를 보여 줌.
3. **휴대폰 인증 앱 대신** vault를 씁니다.
4. vault 앱 → 해당 사이트 항목 편집 → **인증기 키(TOTP)** → QR 스캔 버튼.
5. 폰 카메라로 화면의 QR 스캔.
6. 사이트에 다시 돌아와서, vault가 보여 주는 6자리 코드 입력.
7. 끝.

이제부터 그 사이트 로그인할 때마다 vault가 6자리 코드를 자동으로 보여 줍니다.
자동 입력 사용 시 비밀번호 채워진 다음에 자동으로 6자리 코드도 복사됩니다.

### Q8. 새 사이트에 가입할 때 비밀번호 어떻게 저장하나요?

자동입니다.

1. 회원가입 페이지에서 비밀번호 칸 클릭.
2. vault가 "비밀번호 생성?" 물어봄 → 누름 → 자동 생성·자동 입력.
3. 가입 완료 → 브라우저가 "이 비밀번호 저장할까요?" 물음 → **저장**.

저장이 안 떴으면 vault 앱 열어서 직접 추가:
- + 추가 → 사이트 이름, URL, 사용자명, 비밀번호 입력 → 저장.

### Q9. 아이폰에서 자동 입력이 안 됩니다.

다음 순서로 확인하세요.

1. 아이폰 **설정** → **암호** → **암호 옵션** → **암호 자동 완성** 켜기.
2. 같은 화면 아래에서 **iCloud 키체인** 체크 해제, **Bitwarden** 체크.
3. Bitwarden 앱이 로그인 잠금 해제 상태인지(잠겨 있으면 자동 입력 안 됨).
4. 그래도 안 되면 폰을 한 번 껐다 켜기.

`troubleshooting-ko.md`에 더 자세한 단계가 있습니다.

### Q10. Chrome 브라우저에서 자동 입력이 안 됩니다.

1. 우측 상단 작은 파란 자물쇠 아이콘 보이시나요? 안 보이면 확장이 안 깔린 것 → 다시 설치.
2. 아이콘에 빨간 점이 있으면 잠금 상태 → 누르고 Master Password 입력.
3. 사이트 주소가 vault에 저장된 URL과 다르면 자동 입력이 안 됨.
   → vault에서 항목 편집 → URL 추가 (예: `naver.com`, `nid.naver.com` 둘 다).
4. 그래도 안 되면 자물쇠 아이콘 → "이 페이지에서 자동 입력" 수동 누름.

## 비밀번호 관리

### Q11. Bitwarden과 vault는 같은 건가요? 다른 건가요?

좋은 질문입니다. 이렇게 이해하시면 됩니다.

- **Bitwarden** = 폰·노트북에 까는 **앱 이름**. 비밀번호를 화면에 보여 주는 도우미.
- **vault** = 우리 집 서버 안에 있는 **금고 그 자체**. 비밀번호가 실제로 저장된 곳.

비유:
- Bitwarden 앱은 **자동차의 핸들과 운전석**.
- vault는 **그 핸들을 통해 움직이는 자동차 그 자체**.

다른 회사가 운영하는 Bitwarden 공식 서비스도 있는데, 우리는 그걸 안 쓰고 우리 집 서버를 쓰는 거예요.

### Q12. 사이트에서 비밀번호 바꾸면 vault에도 같이 바뀌나요?

자동은 아닙니다. 다음과 같이 하세요.

1. 사이트에서 평소처럼 비밀번호 변경 절차 진행.
2. 새 비밀번호 만들 때 vault의 "비밀번호 생성" 기능 사용.
3. 변경 완료되면 브라우저가 "비밀번호를 업데이트할까요?" 물음 → **업데이트**.
4. 만약 안 물으면 vault 앱에서 항목 직접 편집 → 새 비밀번호 입력 → 저장.

> 옛날 비밀번호는 굳이 지우지 마시고, 항목 노트 칸에 "2026-05-12 변경 전 비밀번호" 같이 적어 두면 좋아요.

### Q13. 가족 공유 컬렉션과 본인 비밀번호 어떻게 구분되나요?

- **본인 비밀번호** — 소유자 "나(Me)". 본인만 봄. 가족 못 봄.
- **가족 공유** — 소유자 "Realchoice Family", 컬렉션 "가족 공유". 가족 전원이 봄.

비밀번호 추가할 때 **소유자**를 잘 선택하세요. 잘못 넣으면:
- 본인 카카오뱅크를 가족 공유에 → **가족이 본인 통장을 볼 수 있음** (위험!)
- 가족 넷플릭스를 본인 소유로 → 가족이 못 봐서 불편.

헷갈리면 운영자에게 물어 보세요.

### Q14. 호텔 Wi-Fi 같은 임시 비밀번호도 저장해야 하나요?

권장은 안 합니다. 임시 비밀번호는 vault를 어지럽힙니다.

- **자주 쓸 비밀번호** (예: 본인이 자주 가는 카페 Wi-Fi) → 저장.
- **딱 한 번 쓰고 버릴 비밀번호** → 굳이 저장 X. 노트장이나 메모에 잠깐 적었다가 지우기.

### Q15. 비밀번호가 노출된 것 같아요. (예: 해킹 뉴스에서 우리 가입 사이트 이름 봄)

즉시:

1. 그 사이트에서 비밀번호 변경 (vault로 새로 생성).
2. **같은 비밀번호를 다른 사이트에 쓰셨다면**, 그 사이트들도 모두 변경.
   - vault에는 "비밀번호 재사용" 알림 기능이 있어요. 데스크탑 vault → 도구 → 비밀번호 분석.
3. 운영자에게 한 마디 알려 주세요.

## 다른 정보 저장

### Q16. 카드번호도 vault에 저장할 수 있나요?

기술적으로는 가능합니다. + 추가 → 유형 "카드" 선택 → 카드번호, 유효기간, 보안 코드 입력.

**다만 운영자와 한 번 상의하세요.** 우리 가족 정책상 일부 사람만 카드 저장을 허용할 수 있습니다.
가족 공유 컬렉션에는 카드번호 절대 넣지 마세요.

### Q17. 신분증, 여권 사진도 저장 가능한가요?

네. + 추가 → 유형 **보안 메모(Secure Note)** → 노트 칸에 텍스트 + 첨부 파일로 사진 업로드.

이런 자료를 저장하면 편합니다:
- 운전면허증, 주민등록증 사진.
- 여권 정보 페이지 사진.
- 자동차 등록증 사진.
- 의료보험증, 운전면허번호.

**주의**: 첨부 파일은 가족 공유 컬렉션에 넣지 마세요. 본인 항목으로만.

### Q18. Wi-Fi 비밀번호도 저장 가능한가요?

네. + 추가 → 유형 **보안 메모** 또는 **로그인** → 이름 "집 Wi-Fi", 사용자명에 SSID(네트워크 이름), 비밀번호 칸에 비밀번호.

집 Wi-Fi는 가족 공유 컬렉션에 넣어 두면 손님 오셨을 때 보여 주기 편합니다.

## 일반·정책

### Q19. 무료인가요? 매월 돈 내야 하나요?

100% 무료입니다.

- 우리 집 macmini 서버에서 직접 돌아갑니다.
- 다른 회사에 비용을 내지 않습니다.
- 전기료만 듭니다(한 달 몇 천 원).

### Q20. 친구·동료에게 권유해도 되나요?

우리 vault는 **가족 전용**입니다.
친구분이 같은 도구를 원하시면 안내해 주세요.

- 공식 무료 계정: `bitwarden.com` 회원가입 (개인은 평생 무료).
- 본인이 직접 셋업하고 싶으시면 운영자에게 "vault 셋업 가이드" 물어보시면 안내해 드림.

### Q21. 본인이 죽거나 사고가 나면 가족이 비밀번호 어떻게 보나요?

좋은 질문입니다. 두 가지 방법이 있어요.

**방법 1: Emergency Access (긴급 접근)**

1. 데스크탑 vault → 우측 상단 본인 이름 → **긴급 접근(Emergency Access)**.
2. **신뢰할 사람 추가** → 가족 구성원 이메일 입력 → 대기 시간(예: 7일) 설정.
3. 그 가족이 초대 수락.
4. 비상 시 그 가족이 vault에서 "긴급 접근 요청" → 본인이 7일 안에 거부 안 하면 자동 승인 → 본인 vault 전체 열람 가능.

**방법 2: Recovery Code 봉투**

본인 Master Password와 Recovery Code를 적은 종이를 **봉인된 봉투**에 담아 거실 금고나 변호사에게 맡깁니다.
"내가 죽으면 열어라"라는 메모와 함께.

방법 1을 권장합니다. 방법 2는 봉투를 도둑 맞으면 위험합니다.

### Q22. vault 데이터가 우리 집 서버에 있다는데, 백업은 어떻게 되나요?

매일 새벽 3시에 자동 백업됩니다.

- 우리 집 macmini 내부 디스크 → 매일 백업.
- 매주 외부 클라우드(Backblaze B2)에 암호화해서 업로드.
- 분기마다 USB로도 백업해 금고에 보관.

집에 불이 나거나 macmini가 망가져도 비밀번호는 복구 가능합니다.
다만 본인 Master Password는 본인이 외우셔야 하니 잊지 마세요.

### Q23. 회사에서도 vault를 쓸 수 있나요?

회사 Wi-Fi에 있어도 Tailscale 앱만 켜져 있으면 우리 집 vault에 접속됩니다.
회사 보안 정책상 Tailscale 설치가 막혀 있다면 회사 IT 부서에 문의하세요.

## 정말 막힐 때

위 답이 도움 안 되시면 부끄러워 마시고 운영자에게 다음과 같이 연락 주세요.

> "vault에서 이런 문제가 있어요: [상황 설명]. [폰/노트북] 화면에 [메시지]가 떴어요."

화면 사진 한 장 같이 보내 주시면 더 빠릅니다. (다만 화면에 Master Password가 보이면 안 돼요!)

기술적인 트러블슈팅은 별도 문서 `troubleshooting-ko.md`를 보세요.
